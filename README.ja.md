## 🔍 Issue を作成する前に

重複を避け、効率的に Issue を管理するために：

1. **既存の Issue を検索**: 検索バーを使って同様の Issue がないか確認
2. **クローズされた Issue を確認**: すでに解決済みの可能性があります
3. **ドキュメントを読む**: 使用方法の質問の場合があります

📂 **Issue を閲覧**: [すべての Issue を表示](https://github.com/yoima-jp/SmoothThirdPersonCamera-issues/issues)

## 📋 バグ報告に必要な情報

バグの再現と修正を迅速に行うため、以下の情報を含めてください：

### 🎮 環境詳細
| 情報 | 説明 | 必須 |
|-------------|---------|----------|
| **Minecraft バージョン** | 使用しているMinecraftのバージョン | 必須 |
| **Mod ローダー** | Fabric、Forge、NeoForge のいずれか | 必須 |
| **ローダーバージョン** | 使用しているローダーのバージョン | 必須 |
| **Smooth Third Person Camera バージョン** | インストールしているModのバージョン | 必須 |
| **Java バージョン** | 使用しているJavaのバージョン | 必須 |
| **オペレーティングシステム** | WindowsやmacOS、Linuxのバージョン | 必須 |

### 🔄 再現手順
1. 何をしたかを正確に説明
2. 関連する設定や構成を含める
3. 関係する可能性のある他の Mod について言及

### 📁 ログファイル
- **クラッシュレポート**: クラッシュ Issue に必須
- **latest.log**: 競合や警告の特定に役立ちます
- **スクリーンショット/動画**: 問題の視覚的証拠
