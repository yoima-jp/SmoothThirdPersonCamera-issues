## 🔍 Before Creating an Issue

To avoid duplicates and help us efficiently manage issues:

1. **Search existing issues**: Use the search bar to check if your issue already exists
2. **Check closed issues**: Your issue might have been resolved already
3. **Read the documentation**: Some issues might be usage questions

📂 **Browse Issues**: [View All Issues](https://github.com/yoima-jp/SmoothThirdPersonCamera-issues/issues)

## 📋 Essential Bug Report Information

To help us reproduce and fix bugs quickly, please include:

### 🎮 Environment Details
| Information | Description | Required |
|-------------|---------|----------|
| **Minecraft Version** | The version of Minecraft you're using | Required |
| **Mod Loader** | Fabric, Forge, or NeoForge | Required |
| **Loader Version** | The version of your mod loader | Required |
| **Smooth Third Person Camera Version** | The version of the mod you have installed | Required |
| **Java Version** | The Java version you're running | Required |
| **Operating System** | Your OS and version (Windows, macOS, Linux) | Required |

### 🔄 Reproduction Steps
1. Describe exactly what you did
2. Include any relevant settings or configurations
3. Mention other mods that might be involved

### 📁 Log Files
- **Crash Report**: Essential for crash issues
- **latest.log**: Helps identify conflicts and warnings
- **Screenshots/Videos**: Visual evidence of the problem
