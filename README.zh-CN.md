## 📋 Bug 报告必要信息

为了帮助我们快速重现和修复 Bug，请包含：

### 🎮 环境详情
| 信息 | 说明 | 必需 |
|-------------|---------|----------|
| **Minecraft 版本** | 你正在使用的Minecraft版本 | 必需 |
| **Mod 加载器** | Fabric、Forge 或 NeoForge | 必需 |
| **加载器版本** | 你使用的mod加载器版本 | 必需 |
| **Smooth Third Person Camera 版本** | 你安装的mod版本 | 必需 |
| **Java 版本** | 你运行的Java版本 | 必需 |
| **操作系统** | 你的操作系统和版本（Windows、macOS、Linux） | 必需 |

### 🔄 重现步骤
1. 准确描述你做了什么
2. 包含任何相关设置或配置
3. 提及可能涉及的其他 Mod

### 📁 日志文件
- **崩溃报告**：崩溃问题必需
- **latest.log**：帮助识别冲突和警告
- **截图/视频**：问题的视觉证据
